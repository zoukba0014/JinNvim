# JinNvim

## Installation
+ install neovim

```bash
brew install neovim
```

+ For use telescope
```bash
brew install ripgrep
```

+ For use lazygit
```bash
brew install lazygit
```

+ install jinNvim
```bash
git clone https://github.com/zoukba0014/JinNvim.git ~/.config/nvim
```

+ start nvim
```bash
nvim
```
